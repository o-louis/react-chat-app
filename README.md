# react-chat-app

It is a simple chat app like facebookMessenger which is developed in reactjs.
You can create or join a room with an ID.

## How does it works ?

1. To run the server<br>
`cd server`<br>
`npm install`<br>
`npm run start:dev`

2. To run the client<br>
`cd client`<br>
`npm install`<br>
`yarn start`<br>
